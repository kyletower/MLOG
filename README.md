## Structure

```text
mlog/
├── articles/
│   ├── Article-Title/
│   │   ├── summary.md
│   │   ├── notes.md
│   │   └── thoughts.md
├── books/
│   ├── Book-Title/
│   │   ├── summary.md
│   │   ├── notes.md
│   │   └── thoughts.md
├── podcasts/
│   ├── Podcast-Title/
│   │   ├── Episode_1.md
│   │   ├── Episode_2.md
│   │   └── summary.md
├── movies/
│   ├── Movie-Title.md
├── shows/
│   ├── Show-Title/
│   │   ├── Season-1/
│   │   │   ├── Episode-1.md
│   │   │   └── Episode-2.md
│   │   └── summary.md
└── README.md
```
